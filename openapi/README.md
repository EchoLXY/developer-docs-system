# OpenAPI Specification

This directory contains the OpenAPI (Swagger) specification for the Order Management API.

## Purpose

- Provide machine-readable API documentation
- Enable automatic documentation generation
- Improve consistency between implementation and documentation
- Support developer onboarding and integration

## Features

- Defined request/response schemas
- Authentication (JWT Bearer)
- Structured endpoint definitions
- Reusable components

## How to Use

You can visualize this API using:

- Swagger Editor (https://editor.swagger.io)
- Swagger UI
- Postman (import as OpenAPI)

## Design Approach

- Follow RESTful API design principles
- Ensure consistency in naming and structure
- Provide clear examples for all endpoints
- Align documentation with real-world developer usage
